# ghorplayerBot 🎵

رباتِ مستقلِ پخشِ موزیک در ویس‌چتِ گروه‌هایِ تلگرام (فارسی) - جدا از
[ghormanagmentBot](https://github.com/Pedram-Naghib/ghormanagmentBot)، با
سلسله‌مراتبِ نقشِ مشابه (مالکِ ربات > ادمینِ کل > مالکِ اصلیِ گروه > مالکِ ۲ >
ادمین > عادی) اما **بدونِ عضوِ ویژه**، چون اینجا فقط کنترلِ موزیک اهمیت داره
و ویژه هیچ‌وقت به اون دسترسی نداشت.

## قابلیت‌ها
- پخش با ریپلای رویِ فایلِ صوتی/ویدیویی، یا جستجو/لینکِ یوتیوب
- استخرِ چند-یوزرباته (چند گروه هم‌زمان، هر گروه به یک اکانتِ ثابت چسبیده)
- صف، شافل، لوپ (یک آهنگ/کلِ صف)، ولوم، میوت
- هابِ شیشه‌ای با دکمه، با پشتیبانیِ بنرِ اختیاریِ عکس/گیف/ویدیو
- خروجِ خودکار از ویس‌چت بعدِ بیکاری

## نصب و راه‌اندازی
۱. `.env.example` رو کپی کن به `.env` و پر کن (توکنِ بات، دیتابیسِ Supabase، آیدیِ خودت).
۲. `pip install -r requirements.txt`
۳. برای موزیک، راهنمایِ کاملِ `setup.md` رو دنبال کن (ساختنِ سشنِ یوزربات، اضافه کردنش به گروه، و ...).
۴. `python bot.py`

## دستورها
| دسته | دستور |
|---|---|
| موزیک | «پخش» (ریپلای) / «پخش آهنگ &lt;اسم یا لینک&gt;» / «هاب» / «مکث» / «ادامه پخش» / «بعدی» / «پایان پخش» / «شافل» / «یوزربات‌ها» |
| نقش‌ها | «افزودن مالک دو» / «حذف مالک دو» / «افزودن ادمین» / «حذف ادمین» / «مدیران» |
| ادمینِ کل (بات‌واید) | «افزودن ادمین کل» / «حذف ادمین کل» / «لیست ادمین کل» |
| بنر | «ثبت تصویر [کلید]» (مثلاً `music_hub_banner`) - فقط مالکِ ربات/ادمینِ کل |

مالکِ اصلیِ هر گروه خودکار ثبت می‌شه: هرکس ربات رو به اون گروه اضافه کنه.
